## Tema: Venta de Entradas Mundial Qatar 2022
   
Nuestra idea principal para el proyecto integrador de JAVA es desarrollar un sistema de gestión de venta de entradas para el Mundial que se desarrollará el corriente año en Qatar. El sistema contara con el registro e inicio de sesión de los usuarios,  


Con el fin de que los usuarios obtengan una experiencia inolvidable desde el primer día.
